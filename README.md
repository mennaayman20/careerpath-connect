
🚀 Upply | AI-Powered Recruitment Platform
Upply is a modern SaaS platform designed to automate and enhance the recruitment lifecycle using Artificial Intelligence. Built as a Senior Graduation Project, it bridges the gap between talent and opportunities.

 Core Highlights
AI Resume Scoring: Instant feedback and skill-gap analysis for candidates.

Smart Ranking: AI-driven applicant sorting for recruiters.

Modern UI: Clean, modular design with full Dark Mode support.

Secure Auth: Token-based activation and protected routes.

🛠️ Tech Stack
Frontend: React + TypeScript + Vite + Tailwind CSS.

Architecture: Modular (Services, Hooks, Interfaces).
